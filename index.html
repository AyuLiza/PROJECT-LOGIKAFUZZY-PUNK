🧭 Alur Proses Metode Fuzzy Mamdani

Setiap kali nilai input diubah, sistem otomatis menjalankan 4 tahap berikut untuk menghasilkan rekomendasi.

  1. Fuzzifikasi

    Nilai input diubah menjadi derajat keanggotaan (0–1) pada himpunan fuzzy "rendah/sedang/tinggi" menggunakan fungsi segitiga.

  2. Aplikasi Fungsi Implikasi (MIN)

    Untuk tiap aturan, nilai α-predikat dihitung sebagai minimum dari ketiga derajat keanggotaan input.

    α = min(μ_suhu, μ_kelembapan, μ_air)
  3. Komposisi Aturan (MAX)

    Daerah fuzzy keluaran dari semua aturan aktif digabungkan menggunakan operator maksimum.

  4. Defuzzifikasi (Centroid)

    Daerah agregat dihitung titik beratnya untuk menghasilkan nilai akhir crisp dalam kg/Ha (pupuk) dan L/s/Ha (irigasi).

    z* = ∫ z·μ(z) dz / ∫ μ(z) dz

⚙️ Aturan Fuzzy yang Aktif (α > 0)
